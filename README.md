# Funny Text

Convert input text to text with messed up upper and lower case. Feel free to suggest a better name than 'funny-text'.

## Projects setup

```
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

## Run the program

```
python FunnyText.py
```

or

```
./FunnyText.cmd
```
